larch – a do-it-yourself live Arch Linux CD



Quick Start



A quick run through of the steps necessary to build your own larch CD. This should provide you with a working live CD, but to take full advantage of the flexibility offered by the larch build system you will probably need to consult the rest of the documentation.

  • At present larch-5 is still being tested, and is only available from the subversion repository. To fetch the source: 
    svn export svn://svn.berlios.de/larch/trunk
    Then go to the 'trunk/misc' directory and run mklarchpak. This will create a convenient packet for distributiong larch.
  • Unpack the 'larchpak.tar.gz' archive, enter the 'larchpak' directory and run larch-setup.
  • Run './mklarch -h' to get a usage description.
  • You can then try building one of the example profiles, by copying the directory (say 'mini2') from the 'trunk/profiles' directory to the current directory and then running (e.g.) './mklarch -p mini2'. You can also adapt one of the example profiles to your own needs.
  • If you chose a delivered profile, the build should just run, apart from asking for confirmation a few times.
  • For a custom build:
    • Edit 'locale.gen' for your glibc locale needs.
    • Edit 'rcconfx', which contains just the changes to the default /etc/rc.conf. Especially the locale and console keyboard setting may be interesting.
    • Editing the overlay (the set of files which are changed from the default installation) is a bit trickier, as permissions and ownerships must be preserved. Please read the relevant documentation before attempting this.
    • OPTIONAL: If you want ssh access to your live system, see 'ssh access'
    before running mklarch.
  • Another possibility is to use larchify (run './larchify -h' for a usage description) to 'livify' an existing Arch Linux installation, even the currently running one. Running './larchify path/to/installation' should be enough for a simple build, but see 'Making a live CD from an existing Arch installation' for more details.
  • After a while, the iso for the CD should be ready for burning.
  • Alternatively, you can put it on a USB-stick, by passing the '-u' option to 'mklarch'.
  • Share and Enjoy!
